# AoC 2025

## Quick Start

Install dependencies executable:
```sh
bun
```

1. Go on Advent of Code and login
2. Check your Session : Tools -> Application -> Cookie
3. Get your session
4. Save your session in .env :
```
AOC_SESSION=xxxxxxxxxxx
```
5. Generate your first day (downloading input / generating part1 and part2 / open web page of the enigm) :
```sh
bun gen 1
```
6. Run your code for a specific day:
```sh
bun day 1
```
